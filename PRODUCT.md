# Product

## Register

brand

## Users
Experienced software developers (strong React/JavaScript/TypeScript, PHP/Laravel backgrounds) who want to learn backend development in Go.

## Product Purpose
Providing high-quality, comprehensive, and project-based Go backend learning material in Indonesian. The goal is to build a production-grade backend for an online skincare shop and deploy it to AWS, bridging Go concepts with their existing knowledge.

## Brand Personality
Warm, clear, technical, professional, and editorial. Calm and precise guidance without condescension.

## Anti-references
Generic SaaS templates, over-saturated layouts, gradient text, excessive cards, side-stripe borders, and arbitrary numbered/uppercase eyebrows on every section.

## Design Principles
- **Jembatani Pengetahuan**: Always start by connecting Go concepts to React/JavaScript and PHP/Laravel equivalents before defining Go-specific behaviors.
- **Riset & Akurasi Utama**: All code examples, dependencies, and APIs must reflect production-grade patterns (e.g., Go 1.26, chi v5, pgx v5).
- **Proyek Nyata Terarah**: Focus on building a single, tangible skincare online shop project, moving step-by-step from foundations to AWS deployment.

## Accessibility & Inclusion
- Strict adherence to WCAG AA/AAA contrast guidelines.
- Support for system-wide user preferences (e.g., dark/light/sepia themes, font resizing, and `prefers-reduced-motion`).
